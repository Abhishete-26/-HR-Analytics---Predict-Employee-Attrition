# HR Analytics - Employee Attrition Prediction

## 📊 Project Overview
This project analyzes employee attrition using Python and Machine Learning to identify key factors influencing employee turnover.

## 🛠 Tools Used
- Python (Pandas, Seaborn, Scikit-learn)
- Power BI

## 📁 Files Included
- attrition_project.py
- HR_Attrition_Dashboard.pbix
- HR_Attrition_Dashboard.pdf
- HR Analytics report.pdf
- Dataset (.csv)

## 📌 Key Insights
- Salary impacts attrition
- Certain departments have higher attrition
- ML model predicts employee turnover

## 📈 Model Performance
- Accuracy: ~80–85%
- Evaluated using confusion matrix
